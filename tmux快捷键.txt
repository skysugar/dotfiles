| 快捷键                                | 功能说明                    |
| ---------------------------------- | ----------------------- |
| **C-s C-b**                        | 发送前缀键本身（在嵌套 tmux 时使用）   |
| **C-s C-o**                        | 轮换当前窗口的面板布局（在所有面板之间旋转）  |
| **C-s C-z**                        | 挂起当前客户端（暂停 tmux 窗口）     |
| **C-s Space**                      | 切换到下一个布局（布局切换：水平、垂直等）   |
| **C-s !**                          | 将当前面板分离为新的窗口            |
| **C-s #**                          | 列出所有粘贴缓冲区               |
| **C-s $**                          | 重命名当前会话                 |
| **C-s &**                          | 关闭当前窗口                  |
| **C-s '**                          | 通过编号选择窗口                |
| **C-s (**                          | 切换到上一个客户端               |
| **C-s )**                          | 切换到下一个客户端               |
| **C-s ,**                          | 重命名当前窗口                 |
| **C-s .**                          | 移动当前窗口到指定索引             |
| **C-s /**                          | 查看按键绑定说明                |
| **C-s 0–9**                        | 选择编号对应的窗口               |
| **C-s :**                          | 打开命令行模式（输入 tmux 命令）     |
| **C-s ;**                          | 切换到上一个活动面板              |
| **C-s =**                          | 从列表中选择一个粘贴缓冲区           |
| **C-s ?**                          | 列出所有快捷键绑定               |
| **C-s C**                          | 打开 tmux 选项定制界面          |
| **C-s D**                          | 从列表中选择客户端               |
| **C-s E**                          | 让所有面板平均分布               |
| **C-s M**                          | 清除已标记的面板                |
| **C-s [**                          | 进入复制模式（copy-mode）       |
| **C-s ]**                          | 粘贴最近的缓冲区内容              |
| **C-s d**                          | 从当前 tmux 会话中分离（detach）  |
| **C-s f**                          | 搜索面板                    |
| **C-s i**                          | 显示当前窗口信息                |
| **C-s m**                          | 标记/取消标记当前面板             |
| **C-s n**                          | 切换到下一个窗口                |
| **C-s o**                          | 切换到下一个面板                |
| **C-s p**                          | 切换到上一个窗口                |
| **C-s q**                          | 显示面板编号（短暂显示以便选择）        |
| **C-s s**                          | 从列表中选择会话                |
| **C-s t**                          | 显示一个时钟                  |
| **C-s w**                          | 从列表中选择窗口                |
| **C-s x**                          | 关闭当前面板                  |
| **C-s z**                          | 放大/还原当前面板               |
| **C-s {**                          | 交换当前面板与上方面板             |
| **C-s }**                          | 交换当前面板与下方面板             |
| **C-s ~**                          | 显示 tmux 消息日志            |
| **C-s DC**                         | 重置窗口可见区域，跟随光标           |
| **C-s PPage**                      | 进入复制模式并向上滚动一页           |
| **C-s ↑ / ↓ / ← / →**              | 在面板之间移动                 |
| **C-s M-1~5**                      | 设置面板布局（水平、垂直、主从、平铺等）    |
| **C-s M-n / M-p**                  | 切换到有警告（活动提示）的下/上一个窗口    |
| **C-s M-o**                        | 反向旋转面板                  |
| **C-s M-方向键**                      | 面板尺寸调整（步长 5）            |
| **C-s C-方向键**                      | 面板尺寸调整（步长 1）            |
| **C-s S-方向键**                      | 在复制模式中移动视图              |
| **C-s S-Left / Right / Up / Down** | 滚动视图（非移动光标）             |

C-s S-Down  Move the visible part of the window down
